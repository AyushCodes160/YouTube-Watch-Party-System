# Watch Together

Watch YouTube videos together with your friends in real-time.

## Features

- **Real-time Synchronization**: Watch videos in perfect sync with everyone in the room.
- **Group Chat**: Chat with your friends while watching.
- **Easy Sharing**: Share a simple link to invite others.

## Technologies

This project is built with:

- **Vite**
- **TypeScript**
- **React**
- **shadcn-ui**
- **Tailwind CSS**
- **Supabase**

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**

   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. **Install dependencies**

   ```sh
   npm install
   ```

3. **Start the development server**
   ```sh
   npm run dev
   ```
