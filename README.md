# YouTube Watch Party

Watch YouTube together in perfect sync.

## Project overview

This project allows multiple users to watch YouTube videos simultaneously and synchronized in a shared session.

## Technologies Used

- Frontend: React, Vite, Tailwind CSS, shadcn/ui
- Built by AyushCodes160

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the frontend application locally.
